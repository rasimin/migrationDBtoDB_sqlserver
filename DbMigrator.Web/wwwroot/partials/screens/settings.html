Pengaturan Login

Atur apakah aplikasi dapat langsung dibuka atau wajib menggunakan username dan password.

Akses Aplikasi

Disimpan langsung ke auth-config.json.

Minimal 3 karakter.
Minimal 8 karakter. Password lama tidak ditampilkan.

Cara kerja mode akses

Login aktif

Semua halaman, API, dan koneksi SignalR hanya dapat digunakan setelah login.

Bypass login

Aplikasi terbuka langsung tanpa meminta username atau password.

File JSON memuat password dalam teks biasa sesuai konfigurasi lokal. Batasi akses file dan jangan publikasikan kredensial produksi.
