Selamat Datang di DbMigrator

Pilih Migration Job di panel kiri untuk mulai mengelola migrasi.
Setiap job memiliki tab Data Migration dan Object Migration yang menggunakan connection yang sama.

Data Native SQL

Tambahkan script SQL sebagai langkah dalam urutan migrasi data. Gunakan {{SOURCE_DB}} dan {{TARGET_DB}} untuk query antar database di server yang sama.

Pengaturan Objek Tabel

                
Konfigurasi aturan sinkronisasi skema untuk tabel target.

            

            
        

        

            
            

                
                
            

            
            

                

                    
                    

                        
                        
Jika diaktifkan, kolom pada tabel Target yang tidak ada di tabel Source akan dihapus selama migrasi skema tabel dijalankan.

                    

                

                

                    
                    PERINGATAN: Mengaktifkan opsi ini berisiko menghapus data kolom pada database Target jika kolom tersebut tidak ada di database Source. Harap gunakan dengan hati-hati.

                Anda dapat mengetik satu nama tabel atau menulis banyak tabel sekaligus dengan pemisah koma.
